Processo de instalação das bibliotecas C e primeiro teste

Baixando todas as bibliotecas e copiando para o EV3

1- Clonar o repositório

Na pasta EV3DEV criada, ou em alguma outra de sua preferência, clonar o repositório ev3dev-c do GitHub

~$  git clone https://github.com/in4lio/ev3dev-c

2- Criar uma nova pasta no terminal do EV3

Esta pasta será onde as bibliotecas, exemplos e etc serão guardadas

~$  mkdir c-libraries
~$  cd c-libraries

3- Copiar o repositório "ev3dev-c" para o EV3

~$  scp -r /caminho/completo/para/o/arquivo robot@ev3dev.local:caminho/onde/colocar/os/arquivos/ev3

No nosso exemplo seria:

~$  scp -r /home/username/Documentos/EV3DEV/ev3dev-c robot@ev3dev.local:/home/robot/c-libraries

Repare que estamos copiando todas as bibliotecas disponíveis, porém utilizaremos somente a biblioteca C.

4- Vamos compilar as bibliotecas:

~$  cd ev3dev-c/source/ev3
~$  make
~$  sudo make install

5- Teste da biblioteca compilada:

~$  cd ../../eg/light
~$  make
~$  Debug/light
