Configurando conexão SSH e instalando os pacotes básicos

Conectando ao EV3

1- Configurar Conexão

  • No computador Host, com o EV3 conectado via USB, clique no simbolo de rede do Ubuntu e depois em "editar conexões".
  • Após a janela de Conexões de rede abrir, clique em "adicionar" para criar uma nova conexão.
  • Selecione a opção "Ethernet" na caixa de seleção e clique em "criar".
  • Escolha um nome para a sua conexão e, no campo "Endereço MAC clonado" escolha o endereço MAC que inicie com 12:16:53.
  • Vá para a aba "Configurações IPv4" e selecione o método "Compartilhado com outros computadores".
  • Clique em salvar, as configurações do computador host estão prontas.
  • No EV3, selecione "Wireless and Network", e então clique na conexão "Wired".
  • Selecione a opção "Connect" e espere o State mudar de "Connecting" para "Online".
  • Marque a opção "Connect automatically" para conectar automaticamente no futuro.

2- Conectar ao EV3 com SSH

A conexão SSh nos permitirá rodar comandos no EV3 pela rede:

Digite o seguinte comando no terminal:

~$  ssh robot@ev3dev.local

Se for a primeira vez que está realizando a conexão, será solicitado a confirmação de autenticação, basta digitae "yes" no terminal e depois apertar "Enter"

A senha será pedida no terminal. Por padrão ela é: maker.

A partir de agora o terminal onde foram realizados os comandos acima é o terminal do ev3dev.

Instalando pacotes iniciais

1- Instalação do build essential

Digite no terminal do ev3dev os seguintes comandos:

~$  sudo apt-get update
~$  sudo apt-get install build-essential

Ao final da instalação teremos o necessário para fazer os testes futuros.
